Browsing Vocabulary

9/14/2021

Client: A web broswer or other tool that displays web pages to a user.

Server: A computer online that stores web pages and delivers them to users.

Request: A message from a client/broswer using a URL to ask a server for a web page.

Response: A message from the server that includes the code to show the web page.

What Makes a Good Website

9/20/2021

What is HTML?

10/1/21

HTML stands for HyperText Markup Language and it is used to describe the structure of a webpage. HTML was first invented in 1993 by Tim Berners-Lee, and the most widely used version of HTML throughout the 2000s was HTML 4.01. You use HTML by using opening and closing tags, you can put elements like img or p inside of these tags to finish them.

CSS Border Properties

12/8/2021

  • border-width, this CSS property will set the width of the 4 border, this is measured in Thick and Thin etc.
  • border-style, this CSS property will set the style of the the 4 borders, for example a dotted line or a dashed line.
  • border-color, this CSS property will set the color of the 4 borders, for example you could say to change the top border only to red etc.
